What is SSL?

A type of a network protocol.
End-to-End encryption and decryption protocol.
Secure socket layer.
Part of a session layer in the OSI model.
Why SSL?
To provide privacy and reliability between two communicating applications.
Privacy : Connection through encryption and Identification through certificates.
Reliability : Message integrity checks
How it works?
Client and Server consensus about SSL version.
A server certificate signed by unbiased CA like verisign Or Self Signed Certificate
Computation overhead.
